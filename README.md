# Сложно сосредоточиться

Короткое описание

Этот репозиторий содержит статическую одностраничную веб-страницу — набор HTML, CSS и JavaScript файлов для демонстрации интерфейса и стилей.

Быстрый старт

- Открыть `index.html` в браузере.
- Для локального сервера (рекомендуется при работе с относительными путями):

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

Структура проекта

- `index.html` — главная страница.
- `styles/` — стили проекта: `style.css`, `light.css`, `dark.css`, `globals.css`, `variables.css`.
- `scripts/` — скрипты (например, `script.js`).
- `fonts/` — шрифты и `fonts.css`.
- `images/` — изображения и графика.

Разработка

- Для быстрого просмотра используйте расширение Live Server в VS Code или команду выше.
- Вносите изменения в файлы в `styles/` и `scripts/`, затем обновляйте страницу.
- Если добавляете шрифты, обновите `fonts/fonts.css` и подключите в `index.html`.
